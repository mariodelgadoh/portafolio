ERP Enlaces Terrestres Nacionales (ETN)

Python Tkinter SQLite Matplotlib Hashlib

Descripción del Proyecto

Sistema ERP desarrollado para la empresa de transporte Enlaces Terrestres Nacionales (ETN) que integra y gestiona eficientemente las operaciones internas a través de siete departamentos principales: Recursos Humanos, Finanzas, Inventarios, Compras, Proveedores, Ventas y Logística. El sistema cuenta con un usuario Admin con acceso total, así como usuarios generales y responsables para cada área.

Características Principales

  • Recursos Humanos: Contratación de empleados, gestión de pagos, control de personal activo/despedido y cambio de contraseñas.
  • Finanzas: Seguimiento de saldo, registro de transacciones y generación de informes financieros con gráficas.
  • Inventario: Control de productos disponibles y historial de salidas con filtros.
  • Compras y Proveedores: Registro de compras y gestión de proveedores asociados.
  • Ventas: Sistema de venta de boletos con control de 24 asientos por autobús.
  • Logística: Gestión de rutas y horarios.
  • Reportes Generales: Empleados por departamento, ventas totales y gastos totales.

Simulador de Órbitas Planetarias

JavaScript HTML5 CSS3 Métodos Numéricos Newton-Raphson GitHub

Descripción del Proyecto

Programa de Simulación de Órbitas Planetarias y Cálculo del Error Cuadrático Medio. En el ámbito de la astronomía y la exploración espacial, la comprensión precisa de las órbitas planetarias es crucial para diversas aplicaciones, que van desde la investigación científica hasta el diseño de misiones espaciales. Este proyecto se enfoca en el desarrollo de un programa, accesible a través de una página web, que permite simular las órbitas planetarias dentro del sistema solar. La evaluación de la exactitud de las simulaciones se realiza mediante el uso del error cuadrático medio (ECM). El programa emplea métodos numéricos avanzados, como el método de Newton-Raphson, para el cálculo de la anomalía excéntrica y la minimización del error cuadrático.

Características Principales

  • Simulación de órbitas planetarias: Utiliza parámetros orbitales y principios físicos fundamentales para modelar el movimiento de los planetas en el espacio.
  • Cálculo de posiciones: Calcula las posiciones de los planetas en función del tiempo, empleando el método de Newton-Raphson para garantizar la precisión de las órbitas simuladas.
  • Representación visual: Proporciona una representación visual precisa de las órbitas de los planetas alrededor del Sol.
  • Cálculo del Error Cuadrático Medio: Compara las posiciones predichas por el modelo con datos observacionales reales para cuantificar la precisión de la simulación.
  • Ecuación de Kepler: Implementa la ecuación M = E - e·sin(E) para relacionar la anomalía excéntrica con la anomalía media.

Reconocimiento de Dígitos Manuscritos

Python Scikit-learn Random Forest Tkinter NumPy Pillow Scipy

Descripción del Proyecto

Sistema de reconocimiento de dígitos manuscritos que combina el dataset MNIST con muestras propias del usuario para crear un modelo personalizado de alta precisión utilizando Random Forest. Desarrollado por Equipo PI para la materia de Inteligencia Artificial.

Características Principales

  • Preprocesamiento avanzado: Procesamiento de imágenes similar al usado en MNIST con conversión a escala de grises, inversión de colores, recorte automático y redimensionamiento a 28x28 píxeles.
  • Entrenamiento personalizado: Permite capturar muestras propias dibujadas con el mouse para crear un modelo adaptado al estilo de escritura del usuario.
  • Interfaz gráfica intuitiva: Dispone de 5 cuadros independientes para dibujar dígitos con botones para guardar muestras, limpiar y reconocer.
  • Métricas detalladas: Genera reportes completos con precisión general, matriz de confusión, métricas F1-score y análisis de confusiones entre dígitos.

MusicArtists

Vue.js 3 Node.js Express MongoDB JWT Vuex REST API

Descripción del Proyecto

MusicArtists es una plataforma web completa para la gestión y reproducción de música, diseñada para conectar a los usuarios con sus artistas, álbumes y canciones favoritas. Se trata de un proyecto full-stack que combina un frontend interactivo desarrollado con Vue.js 3 y un backend robusto construido con Node.js, Express y MongoDB como base de datos NoSQL.

Características Principales

  • Reproductor persistente por usuario: Cada usuario tiene su propia cola de reproducción y estado de reproducción individual que se guarda al cerrar sesión.
  • Panel de administración completo: Dashboard con estadísticas en tiempo real y operaciones CRUD sobre artistas, álbumes, canciones y usuarios.
  • Sistema de autenticación: Implementa JWT, contraseñas hasheadas con bcrypt y validación de roles para proteger rutas administrativas.
  • Manejo inteligente de imágenes: Las canciones heredan automáticamente la portada del álbum, mientras que los sencillos mantienen su propia imagen personalizada.
  • Diseño responsive: Interfaz completamente adaptable a móviles, tablets y desktop con transiciones suaves y feedback visual inmediato.

HTML5 CSS3 JavaScript Fetch API PokéAPI Vercel GitHub React

Descripción del Proyecto

Este proyecto es una aplicación web interactiva tipo Pokédex, que permite consultar información detallada de distintos Pokémon de manera dinámica. La aplicación consume datos desde una API externa (PokéAPI) para mostrar información como nombre, imagen, habilidades, tipos y otras características relevantes de cada Pokémon.   • Despliegue en la nube: Aplicación desplegada y accesible en Vercel para demostración en línea.

Banderas del Mundo - Aplicación Web Interactiva

HTML5 CSS3 JavaScript API REST Vercel GitHub

Descripción del Proyecto

Esta aplicación web interactiva permite explorar y visualizar banderas de distintos países del mundo. La aplicación consume una API externa (REST Countries API) para obtener información actualizada sobre los países, incluyendo sus banderas, capitales, poblaciones, idiomas y otras características relevantes. El objetivo principal del proyecto es ofrecer una experiencia visual atractiva y dinámica, donde el usuario pueda interactuar con los datos en tiempo real. Sirve como práctica para el consumo de APIs, manipulación del DOM y desarrollo de interfaces modernas en el frontend.

Características Principales

  • Exploración de países: Visualiza todos los países del mundo con sus respectivas banderas e información relevante.
  • Búsqueda en tiempo real: Filtra países por nombre de forma rápida y precisa mientras escribes.
  • Filtrado por región: Filtra países por continente (África, América, Asia, Europa, Oceanía).
  • Información detallada: Muestra capital, población, región, idiomas, monedas y dominio del país.
  • Diseño responsivo: Interfaz completamente adaptable a dispositivos móviles, tablets y desktop.
